# 02 — Custom Header (Standalone)

> Build a **dedicated** header for this page only. **Do not** include the main Belk Body Lab
> navigation menu. The header carries the logo and a single primary CTA.

## Layout
- **Left:** Belk Body Lab logo (links to `#top` of this page only — NOT to the main site).
- **Right:** One CTA button.

## Header CTA
- **Button label:** `View COA Reports`
- **Destination:** `https://99puritypeptides.com/certificates`
- **Attributes:** `target="_blank" rel="noopener noreferrer nofollow" class="cta-coa" data-cta="header-view-coa"`

## Optional eyebrow strip (above logo, full-width, thin)
```
Analytical Transparency · Verified Documentation · Research-Use-Only Standards
```

## Notes
- No search bar, no cart, no account, no dropdown menus.
- Header should be sticky on scroll so the COA CTA stays reachable on long-scroll mobile sessions.
- Keep header height minimal so the hero stays above the fold.
